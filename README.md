# 도장 현장작업 모바일 PWA v7

기준 URL: https://script.google.com/macros/s/AKfycbxiU6pHyhoT7qWBhFq2GZyGaVwMahmwdalOXL0TjoKoY1nXH_kXDFpy4CijyZdLQqn1/exec

## 기능
- 현장 작업자용 간단 입력
- 하도 입력: 작업자/품번/LOT/환경조건/하도두께 저장
- 상도 입력: 상도 입력 대상 검색 후 선택, 총두께/색차/광택 입력 저장
- 사진 첨부, 작업자 관리, 불량/조치 입력은 모바일 화면에서 제외

## 적용
GitHub `paint-mobile-app` 저장소에 파일 전체를 업로드합니다.
서비스워커 캐시명은 `paint-mobile-v7-primer-topcoat-simple`입니다.

- 피도물 표면온도 입력과 환경판정 표시를 작업 정보 영역으로 이동
