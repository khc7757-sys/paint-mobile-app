# 도장 현장작업 모바일 PWA v4

## 변경 내용
- 작업자 관리 영역 제거 상태 유지
- 도장업체명 표시 제거 상태 유지
- 대기온도/상대습도는 기존 날씨 자동 조회값 사용
- 피도물 표면온도 입력칸 추가
- 이슬점온도 자동 표시
- 이슬점차이 = 피도물 표면온도 - 이슬점온도 자동 계산
- 환경판정 자동 계산

## 환경판정 기준
- 피도물 표면온도 43℃ 이하
- 이슬점차이 3℃ 이상
- 두 조건을 모두 만족해야 적합

## 적용 방법
GitHub `paint-mobile-app` 저장소에 이 ZIP 안의 파일 전체를 덮어쓰기합니다.
모바일 주소는 기존과 동일합니다.

https://khc7757-sys.github.io/paint-mobile-app/

적용 후 예전 화면이 보이면 휴대폰에서 기존 PWA 앱을 삭제 후 다시 설치하거나 브라우저 캐시를 삭제합니다.
